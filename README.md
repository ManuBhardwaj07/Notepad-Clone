# 📝 Notepad-Clone

A fully functional desktop notepad application built with **Java Swing**. This project replicates the core utility of a standard text editor with a clean, intuitive interface.

## 🚀 Features

* **File Operations:** Create New files, Open existing ones, and Save your progress.
* **Text Editing:** Standard Cut, Copy, and Paste functionality.
* **Formatting:** Options to adjust text appearance and layout.
* **User Interface:** Built using Java Swing for a native desktop feel.

## 🛠️ Built With

* **Java** - Core programming language.
* **Swing/AWT** - For the Graphical User Interface (GUI).
